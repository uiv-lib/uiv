## Build Setup

```bash
# install dependencies
npm install

# serve docs with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build docs
npm run build-docs

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
